### 營收分析

原始的營業收入歷史資料來源`mops.twse.com.tw`已經失效，並且以非常嚴格的方式**禁止任何爬蟲手段**取得資料，違者會被封鎖IP禁止訪問網站（如下圖）。

![](_bug_revenue_api_blocked.png)

如上述所示，原始的歷史營收資料取得方式已經確定不可行。

#### 證交所Open API

只能免費取得**最近一個月**的營業收入資訊，歷史資料不足。

#### FinMind API

要實現原有功能似乎僅剩高昂費用的正式第三方金融API服務。

評估此方案的相關成本太高，下一步將朝向移除營收分析功能為主。

#### 移除營收分析功能

確保營收分析的相關邏輯如何影響整個APP流程。

關閉營收API的同時保持原有的APP操作流程不變。
